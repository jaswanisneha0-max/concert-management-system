# Concert Management System 🎵

A PHP & MySQL based web application to manage concerts, bookings, and users.

## Features
- Admin Login
- Add / Edit / Delete Concerts
- User Registration & Login
- Ticket Booking System
- MySQL Database Integration

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## How to Run
1. Install XAMPP
2. Copy project into htdocs
3. Import database.sql into phpMyAdmin
4. Open localhost/project-name
